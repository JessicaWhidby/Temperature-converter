OBJECTIVE
This was a class project when I was very new to learning Java. I was also taking a chemistry class at the time, so this project has some extra use for me as I could convert temperatures to and from Kelvin for use in lab writeups.

STEPS TAKEN
First off, I knew I had to take input from the user, in the form of both the starting and ending unit of measurement and the starting measurement itself. I also added in some simple input validation/error handling in case the user did not enter appropriate answers, such as converting Kelvin to Kelvin or Kelvin to ABCD. Then, I had to actually code the math for each potential conversion. Finally, it outputs the converted temperature.

KEY TAKEAWAYS
I learned about how to create a useful program that has applications outside of "this was a school project." I also learned how relationships between methods work and how to handle incorrect input. Overall, a great learning experience in a relatively simple tool.
